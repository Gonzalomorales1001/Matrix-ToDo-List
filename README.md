# Matrix Style To-Do List

This is a To-do List with matrix style.

## Functionalities

- Add Tasks
- Delete Tasks
- Edit tasks
- Mark as done
- Change 'matrix' color theme

> Note: All your changes will be saved in your local storage

### Bulit with

- HTML5
- CSS Custom Properties
- CSS FlexBox
- Javascript Vanilla
